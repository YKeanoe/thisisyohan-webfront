## Installation

```bash
$ npm install && npm build
```

## Running the app

```bash
# development
$ npm run dev
```

## Deployment & Environments

App is running in `pm2` fork mode. And requires **manual deployment**.

```bash
# git pull, install, build & restart server
bash /home/deploy-frontend.sh
```
